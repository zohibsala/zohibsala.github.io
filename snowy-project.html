Snowy

Overview

Snowy is a 3D environment project developed in Unity that explores game engine rendering techniques, focusing on materials, lighting, shaders and optimisation. The project is built around a winter themed outdoor and indoor scene, designed to demonstrate how different rendering systems work together to create a visually engaging and efficient environment.

The main goal was to understand how textures, lighting and advanced shader techniques can improve realism while maintaining performance. The project combines multiple lab tasks into a single cohesive scene that can be explored using a third person controller.

My Role

I was responsible for creating the full environment, including materials, lighting setups, shader effects, terrain design, and optimisation techniques. I also implemented particle systems and a third person controller to test the scene in real time.

Key Features

  • Snowy environment with terrain using multiple texture layers such as snow, grass, dirt and gravel
  • Interior and exterior scenes with consistent lighting and mood
  • Use of Normal Maps, Metallic Maps and other material properties to improve realism
  • Multiple lighting setups including directional, point and spot lights
  • Baked lighting system to improve performance and visual quality
  • Shader Graph material with custom effects such as Fresnel highlights
  • PBR shader with animated effects using nodes like Time and Remap
  • Particle systems for snow and fire effects
  • Skybox variations to create different moods such as sunset, night and snowy
  • Third person controller to explore the environment and test scale

Technical Implementation

I developed the scene using Unity's Material Editor and lighting systems to create realistic surfaces and atmosphere. Materials were improved using Normal Maps, Metallic Maps, Occlusion and Smoothness settings, allowing objects to have depth and more believable surface detail.

Lighting played a major role in the project. I used directional light to simulate sunlight and moonlight, point lights for local sources such as lamps and torches, and spot lights to highlight specific areas. I also implemented baked lighting for static objects, which improved performance while maintaining high quality lighting.

Shader Graph was used to create custom materials, including a Fresnel effect on a streetlamp to highlight edges and improve visual appeal. I also created a PBR shader with animated behaviour, making parts of the lamp glow and change over time using nodes such as Multiply, Time and Remap.

The terrain system was built using multiple texture layers and custom brushes to create a detailed snowy landscape. A particle system was added to simulate snowfall, while additional particle effects were used to create fire for torches and campfires.

To test the environment in real time, I added a third person controller, allowing movement around the scene and helping evaluate object scale, lighting and overall immersion.

Challenges

One of the main challenges was understanding how different texture maps work together. At the beginning, materials appeared flat because I did not fully understand Normal Maps, Metallic Maps and Height Maps. Through experimentation and tutorials, I learned how to balance these settings to achieve a realistic look.

Lighting was also difficult at first, especially learning how each light type affects the scene. It took time to understand how to combine directional, point and spot lights effectively without overexposing or flattening the environment.

Another challenge was optimisation. High quality textures and lighting can affect performance, so I had to adjust texture resolutions and use baked lighting where possible to keep the scene efficient.

Shader Graph was completely new to me, so learning how nodes connect and affect materials required trial and error. However, once understood, it became a powerful tool for creating custom effects.

Outcome

This project helped me build a strong understanding of game engine rendering techniques, including materials, lighting, shaders and optimisation. I learned how to balance visual quality with performance and how different systems in Unity work together to create immersive environments.

It demonstrates my ability to design and develop a complete 3D scene, apply advanced rendering techniques, and experiment with creative solutions such as shader effects and particle systems.
