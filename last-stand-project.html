Last Stand

Overview

Last Stand is a first person zombie survival shooter developed in Unreal Engine 5 as part of my university coursework. This project marked my first experience working with Unreal Engine and building a complete game from scratch.

The game is built around a wave based survival system where players must defend themselves against increasingly difficult waves of enemies. The focus was on creating a fast paced and engaging gameplay loop supported by AI driven enemies and responsive player mechanics.

My Role

I was responsible for designing and implementing the core gameplay systems, including combat mechanics, enemy AI behaviour, UI systems and audio integration.

Key Features

  • Wave based enemy spawning system with increasing difficulty
  • AI navigation using NavMesh to actively track and chase the player
  • Shooting and reloading system for player combat
  • Ammo pickup system to support progression and balance
  • User interface including health system, ammo counter and wave tracker
  • Damage feedback system using screen effects to improve player awareness
  • Audio system using attenuation and reverb for realistic sound behaviour

Technical Implementation

I developed the core gameplay systems using Blueprint scripting in Unreal Engine. This included implementing player combat mechanics such as shooting and reloading, supported by an ammo system and pickup logic to maintain gameplay flow.

Enemy behaviour was built using AI navigation systems, allowing zombies to dynamically locate and chase the player. This created constant pressure and reinforced the survival aspect of the game.

I also designed and implemented a functional user interface displaying player health, ammunition and wave progression. A visual damage feedback system using a blood overlay effect was added to enhance immersion and player awareness.

Audio was integrated using attenuation and reverb systems to ensure sounds reacted realistically based on distance and environment. Character animations were sourced from Mixamo, then refined and implemented to ensure smooth and believable movement within the game.

Challenges

As this was my first time working with Unreal Engine, one of the main challenges was learning the engine while developing under time constraints. This required quick adaptation, problem solving and prioritisation to ensure that all core systems were implemented and the game remained fully playable.

Outcome

This project provided me with a strong foundation in Unreal Engine development, including Blueprint scripting, AI systems, gameplay mechanics, UI implementation and audio design.

It demonstrates my ability to take a project from concept to a complete and playable experience while working across multiple areas of game development.
